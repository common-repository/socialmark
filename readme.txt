== Description ==
<p>SocialMark is a WordPress plugin that allows adding overlay/watermark to social media post link share preview. It gives the option to use different images for social media (Twitter, Facebook) and add overlay/watermark to the preview. The plugin enables 2 different watermark positions(9 in premium version) to set overlay at different places on the post image. There are two settings options: universal settings for all posts and pages, post-wise settings for individual page/post. SocialMark also works with Yoast SEO and RankMath plugin's open graph settings. Add sponsored image on your social media news/post preview easily.</p>

<h3>Installation:</h3>
<p>
<ul>
<li>From the WP admin panel, click "Plugins" -> "Add new"</li>
<li>In the browser Click on "Upload Plugin"</li>
<li>Choose the "socialmark" plugin zip file and click "Install Now"</li>
<li>Activate the plugin</li>
<li>Go to SocialMark Plugin Settings or Click "SocialMark" from admin menu</li>
<li>Add Overlay</li>
<li>Select plugin options</li>
<li>Save Changes</li>
<li>You are ready to go</li>
<li>Alternatively, you can manually upload the plugin to your wp-content/plugins directory</li>
<li>Don't forget to add feature image to your post </li>
<li>Easy way to test social media open graph preview https://www.opengraph.xyz/</li>
</ul>
